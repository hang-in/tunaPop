# tunaPop

tunaPop은 macOS에서 텍스트를 선택했을 때 팝업 창을 통해 AI 기반 액션을 빠르게 수행할 수 있도록 돕는 PopClip 스타일의 macOS 네이티브 애플리케이션입니다. 로컬 Ollama 호환 API를 활용하여 네트워크 연결에 의존하지 않고 안전하고 빠르게 텍스트 요약, 설명, 번역 등의 작업을 처리합니다.

## 주요 기능

- **네이티브 UX**: 마우스 드래그 완료를 감지하여 선택된 텍스트 주변에 직관적인 액션 팝업을 표시합니다.
- **AI 액션**: 요약(Summarize), 설명(Explain), 번역(Translate) 등 자주 사용하는 템플릿 프롬프트를 원클릭으로 실행합니다.
- **로컬 LLM 연동**: Ollama(또는 Ollama 호환 API) 서버와 연동하여 로컬 환경에서 모델(기본값: `llama3.2`)을 실행하므로 개인정보가 외부에 유출되지 않습니다.
- **보안 및 권한 관리**: 시스템 키체인(Keychain)을 사용하여 안전하게 API 토큰을 보관하며, 손쉬운 사용(Accessibility) 및 입력 모니터링(Input Monitoring) 권한 획득 상태를 메뉴 막대에서 직관적으로 제어할 수 있습니다.

## 시작하기

### 요구 사항

- macOS 14.0 이상 (Swift 5.9+, AppKit + SwiftUI)
- 로컬 또는 원격 Ollama 서버 (기본값: `http://localhost:11434`)

### 빌드 및 실행

프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 앱을 빌드하고 실행할 수 있습니다.

```bash
swift run TunaPop
```

> **참고**: 마우스 드래그 및 텍스트 선택을 감지하기 위해 처음 실행 시 macOS '손쉬운 사용(Accessibility)' 권한 승인이 필요합니다.

## 설정

메뉴 바 아이콘을 클릭하고 **Settings**를 선택하여 다음 설정을 조정할 수 있습니다.

- **Endpoint**: Ollama API 주소 (기본값: `http://localhost:11434`)
- **API Token**: 프록시나 Hosted Ollama 등을 사용할 때 필요한 인증 토큰 (입력 시 시스템 키체인에 안전하게 저장됩니다)
- **Model**: 사용할 언어 모델 선택 및 커스텀 모델 수동 입력 지원
